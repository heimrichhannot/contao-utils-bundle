static insertBeforeKey(array $array, string $key, string $newKey, mixed $newValue)

Insert a new entry before an specific key in array.

If key not exist, the new entry is added to the end of the array. Array is passed as reference.

Usage example: contao config.php to make your hook entry run before another.

Parameters

array $array Array the new entry should inserted to
string $key The key where the new entry should be added before
string $newKey The key of the entry that should be added
mixed $newValue The value of the entry that should be added
