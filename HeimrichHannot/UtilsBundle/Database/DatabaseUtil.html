Process a query in pieces, run callback within each cycle.

Parameters

string $countQuery The query that count the total rows, must contain "Select COUNT(*) as total"
string $query The query, with the rows that should be iterated over
callable $callback A callback that should be triggered after each cycle, contains $arrRows of current cycle
string $key The key of the value that should be set as key identifier for the returned result array entries
int $bulkSize The bulk size

Return Value

bool|int False if nothing to do, otherwise return the total number of processes entities

Bulk insert SQL of given data.

Parameters

string $table The database table, where new items should be stored inside
array $data An array of values associated to its field
array $fixedValues A array of fixed values associated to its field that should be set for each row as fixed values
mixed $onDuplicateKey null = Throw error on duplicates, self::ON_DUPLICATE_KEY_IGNORE = ignore error duplicates (skip this entries), self::ON_DUPLICATE_KEY_UPDATE = update existing entries

int $bulkSize The bulk size
string $pk The primary key of the current table (default: id)
